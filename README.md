# SyntheticMemories

A memory system with episodic, semantic, and procedural memory types, inspired by human memory architecture.

## Overview

SyntheticMemories implements a biologically-inspired memory system with:
- **Episodic memory**: Specific events and experiences
- **Semantic memory**: Facts and concepts
- **Procedural memory**: Skills and how-to knowledge
- **Memory links**: Associative connections between memories
